# Processed Data

此資料夾放 Notebook pipeline 的輸出結果。

## Files

- `features.csv`：Notebook1 產生的 rolling statistics、lag、rate 與 ratio features。
- `baseline_anomaly_flags.csv`：Notebook2 產生的 baseline anomaly flags。
- `spc_results.csv`：Notebook3 產生的 SPC control chart results。
- `ml_anomaly_scores.csv`：Notebook4 產生的 unsupervised ML anomaly scores。
- `raw_alerts.csv`：Notebook5 產生的低階 raw alerts。
- `reduced_alerts.csv`：Notebook5 聚合後的 reduced alerts。
- `forecast_results.csv`：Notebook6 產生的 forecast 與 early warning results。
- `rca_events.csv`：Notebook7 產生的 RCA candidate events。
